# Singular-Plural.JS

> A Simple Library to convert singular to plural with all the grammar rules

## Install

    npm i singular-plural-js

## Import

    import Plural from 'singular-plural-js'

## Use

    console.log(Plural(['ray', 'city', 'truss', 'cactus', 'phenomenon']));
